# Демонстрация электронного дневника
# Для запуска дневника вам необходимо установить npm и написать вот так:
# для разработчиков (если вы уже установили все, что вам нужно)
<ul>
    <li>npm run dev</li>
    <li>php artisan serv</li>
</ul>

## в противном случае сделайте это
<ul>
    <li>npm install</li>
    <li>npm run dev</li>
    <li>php artisan serv</li>
</ul>
