Мини документация по запуску электроного дневника (демо)

для запуска дневника, вам нужно в консоле xamp (я просто использовал xamp) 
перейти в путь где находится дневник и в консоле прописать:
[npm install] (если не установлен npm) (если не ошибаюсь командой)
в первую консоль: npm run dev
во вторую консоль: php artisan serv
Готов, сайт должен работать!
